## üöÄ Getting Started

Wanna poke around or run these projects yourself? It's easier than falling off a log!

1.  **Clone this repo:**
    ```bash
    git clone https://github.com/toko-stark/100-days-of-projects.git
    ```
2.  **Navigate to any project directory:**
    ```bash
    cd 100-days-of-projects/01-RockPaperScissors
    ```
    (Replace `01-RockPaperScissors` with the project you want to check out!)
3.  **Open `index.html`:**
    Just drag and drop the `index.html` file into your favorite web browser, or right-click and "Open with Live Server" if you're rocking VS Code.
    ```bash
    # Example for opening in a browser (macOS)
    open index.html
    ```

And voil√†! You're good to go.
